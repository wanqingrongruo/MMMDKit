# 示例

示例目录包含通过 Swift Package Manager 接入 MMMDKit 的原生 App 源码骨架。

## iOSDemo

在 Xcode 中创建新的 iOS UIKit App，添加本地 `MMMDKit` package，然后使用 `Examples/iOSDemo` 中的源码。

需要引入：

- `MMMDParserCmark`
- `MMMDUIKit`

## macOSDemo

在 Xcode 中创建新的 macOS AppKit App，添加本地 `MMMDKit` package，然后使用 `Examples/macOSDemo` 中的源码。

需要引入：

- `MMMDParserCmark`
- `MMMDAppKit`
